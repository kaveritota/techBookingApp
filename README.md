# Technician Booking Application

## Tech Stack
- Frontend: React (Vite), JavaScript, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB

## Features
- User selects technician category
- Weekly calendar view
- Book and cancel technician slots
- Technician dashboard with bookings
- One booking per time slot
- Category-based filtering

## Setup Instructions

### Backend
```bash
cd server
npm install
npm run dev
